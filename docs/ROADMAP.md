## Phase 2 — School pilots (next 6 months)

### EduLens AI (primary)

- Teacher lesson-prep pilot with 2–3 volunteer tutors
- Class-level mistake pattern dashboard (anonymous aggregates)
- Rubric alignment review with one department head

### CampusBot AI

- Robotics CCA workshop using simulator + metrics debrief
- Custom wing map from school floor plan (pilot layout)

### YouthMentor AI

- Counsellor review of safety walkthrough + policy docs
- Anonymous wellbeing insights pilot (no chat transcripts)

## Phase 3 — Product hardening (6–12 months)

| Area | Goal |
|------|------|
| **Accounts** | Optional school SSO for class dashboards (EduLens) |
| **Data** | PDPA-documented consent flows; export/delete at scale |

| **Insights** | School B2B anonymous dashboards (YouthMentor + EduLens) |

## Commercial model (summary)

See [PRICING.md](PRICING.md) and [USER_SURVEY.md](USER_SURVEY.md).

- **B2C:** Freemium limits + premium coaching packs (YouthMentor)
- **B2B:** School site licence for lesson prep + wellbeing insights (aggregates only)
- **Social impact:** Teacher-augmenting AI, not replacement; crisis guardrails by design

| Metric | Target |

| Safety demo comprehension | Counsellor confirms crisis-only behaviour |
| CampusBot reproducibility | Metrics match benchmark table within ±10% |
